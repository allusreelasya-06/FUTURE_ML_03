# 📄 Resume / Candidate Screening System
### Future Interns - Machine Learning Internship | Task 3

## 📌 Project Overview
An ML system that automatically screens and ranks 
resumes based on a given job role using NLP 
and cosine similarity.

## 🛠️ Tools & Technologies
- Python
- NLTK
- Scikit-learn (TF-IDF, Cosine Similarity)
- Pandas & NumPy
- Matplotlib
- Jupyter Notebook

## 📊 Features
- Resume text cleaning & preprocessing
- Skill extraction & matching
- Candidate ranking based on job fit
- Skill gap identification
- Data visualization

## 📈 Results
- Best Match: Alice (74.95%)
- Total Candidates Screened: 5
- Job Role: Machine Learning Engineer
